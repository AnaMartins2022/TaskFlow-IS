# TaskFlow-IS

Projeto de **Integração de Sistemas** baseado em microserviços.

## Tecnologias e Metodologia
- **Arquitetura:** Microserviços (Auth, Task, Notification).
- **Vibe Coding:** Desenvolvimento acelerado com auxílio de IA para modelação de dados e diagramação.
- **Comunicação:** REST (Síncrona) e Message Broker (Assíncrona).
